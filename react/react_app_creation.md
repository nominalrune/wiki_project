# creating a React App

## base creation
- TypeScript
	```powershell
	create-react-app PROJECT_NAME --template typescript
	```
- PWA
	```powershell
	create-react-app PROJECT_NAME --template cra-template-pwa-typescript
	```
- Next
	```powershell
	create-next-react-app PROJECT_NAME
	```
